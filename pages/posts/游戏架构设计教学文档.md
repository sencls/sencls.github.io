## 🎯 核心设计思想

### 场景模式 (Scene Pattern)

通过抽象场景基类实现游戏不同状态之间的平滑切换：

```cpp
// scene.h - 抽象场景基类
class Scene {
public:
    virtual void on_enter() {}                      // 进入场景
    virtual void on_update(int delta) {}            // 每帧更新
    virtual void on_draw(const Camera& camera) {}   // 渲染
    virtual void on_input(const ExMessage& msg) {}  // 输入处理
    virtual void on_exit() {}                       // 退出场景
};
```

**核心优势：**
- ✅ **解耦**：每个场景逻辑独立
- ✅ **可扩展**：新增场景只需继承
- ✅ **生命周期管理**：统一的状态转换

### 游戏循环架构

```cpp
while (true) {
    // 1. 计算帧时间
    DWORD frame_start_time = GetTickCount();

    // 2. 处理输入
    while (peekmessage(&msg)) {
        scene_manager.on_input(msg);
    }

    // 3. 更新逻辑（使用 Delta Time）
    DWORD delta_tick = current_tick_time - last_tick_time;
    scene_manager.on_update(delta_tick);

    // 4. 渲染画面（双缓冲）
    cleardevice();
    scene_manager.on_draw(main_camera);
    FlushBatchDraw();

    // 5. 帧率控制（60 FPS）
    if (frame_delta_time < 1000 / FPS)
        Sleep(1000 / FPS - frame_delta_time);
}
```

**设计亮点：**
- 固定 60 FPS 帧率
- Delta time 机制确保不同帧率下速度一致
- 双缓冲消除闪烁

### 场景管理器

```cpp
class SceneManager {
public:
    enum class SceneType { Menu, Game, Selector };

    void set_current_scene(Scene* scene) {
        current_scene = scene;
        current_scene->on_enter();  // 触发进入回调
    }

    void switch_to(SceneType type) {
        current_scene->on_exit();   // 离开当前场景
        // ... 切换逻辑
        current_scene->on_enter();  // 进入新场景
    }

private:
    Scene* current_scene = nullptr;
};
```

**设计思想：**
- 单一职责：只负责场景切换
- 生命周期管理：确保正确的初始化和清理

### 动画系统

```cpp
class Animation {
public:
    void set_atlas(Atlas* new_atlas) { atlas = new_atlas; }
    void set_interval(int ms) { interval = ms; }

    void on_update(int delta) {
        timer += delta;
        if (timer >= interval) {
            timer = 0;
            idx_frame++;
            if (idx_frame >= atlas->get_size()) {
                idx_frame = is_loop ? 0 : atlas->get_size() - 1;
                if (!is_loop && callback)
                    callback();  // 动画结束回调
            }
        }
    }

private:
    Atlas* atlas = nullptr;
    int timer = 0, interval = 0;
    int idx_frame = 0;
    bool is_loop = true;
    std::function<void()> callback;
};
```

### 计时器系统

```cpp
class Timer {
public:
    void set_wait_time(int val) { wait_time = val; }
    void set_one_shot(bool flag) { one_shot = flag; }
    void set_callback(std::function<void()> cb) { callback = cb; }

    void on_update(int delta) {
        if (paused) return;
        pass_time += delta;
        if (pass_time >= wait_time) {
            if ((!one_shot || (one_shot && !shotted)) && callback)
                callback();
            shotted = true;
            pass_time = 0;
        }
    }

private:
    int pass_time = 0, wait_time = 0;
    bool paused = false, shotted = false, one_shot = false;
    std::function<void()> callback;
};
```

**应用场景：**
```cpp
// 攻击冷却
timer_attack_cd.set_wait_time(500);
timer_attack_cd.set_one_shot(true);
timer_attack_cd.set_callback([&]() { can_attack = true; });
```

### 摄像机系统

class Camera {
public:
    void shake(float strength, int duration) {
        is_shaking = true;
        shaking_strength = strength;
        timer_shake.set_wait_time(duration);
        timer_shake.restart();
    }

    void on_update(int delta) {
        timer_shake.on_update(delta);
        if (is_shaking) {
            position.x = (-50 + rand() % 100) / 50.0f * shaking_strength;
            position.y = (-50 + rand() % 100) / 50.0f * shaking_strength;
        }
    }

## 🎨 设计模式应用

### 状态模式 (State Pattern)

通过不同的动画对象表示角色状态：

```cpp
Animation* current_animation;

// 状态切换
if (direction != 0)
    current_animation = is_facing_right ? &animation_run_right : &animation_run_left;
if (is_attacking_ex)
    current_animation = is_facing_right ? &animation_attack_ex_right : &animation_attack_ex_left;
if (hp <= 0)
    current_animation = last_hurt_direction.x < 0 ? &animation_die_left : &animation_die_right;
```

### 观察者模式 (Observer Pattern)

计时器回调机制：

```cpp
timer_attack_cd.set_callback([&]() {
    can_attack = true;  // 冷却结束
});

timer_invulnerable.set_callback([&]() {
    is_invulnerable = false;  // 无敌时间结束
});
```

### 工厂模式 (Factory Pattern)

角色选择场景创建玩家实例：

```cpp
switch (player_1_choice) {
case Choice::Peashooter:
    player_1 = new PeashooterPlayer();
    break;
case Choice::Sunflower:
    player_1 = new SunflowerPlayer();
    break;
}
```

### 组件模式 (Component Pattern)

Player类整合多个独立组件：

```cpp
class Player {
    Animation animation_idle_left;      // 动画组件
    Timer timer_attack_cd;              // 计时器组件
    std::vector<Particle> particle_list; // 粒子组件
    Vector2 position;                   // 物理组件
};
```
